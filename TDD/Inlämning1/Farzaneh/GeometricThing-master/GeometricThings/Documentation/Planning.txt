## Requirements:
  * Interface class with GetArea and GetPerimeter that each object must implement the methods.
  
  * Square class:
    - constructor that sets parameters
    - public float GetPerimeter();
    - public float GetArea();

  * Rectangle class:
    - constructor that sets parameters
    - public float GetPerimeter();
    - public float GetArea();

  * Equilateral triangle class:
    - constructor that sets parameters
    - public float GetPerimeter();
    - public float GetArea();

  * Circle class:
    - constructor that sets parameters
    - public float GetPerimeter();
    - public float GetArea();

  * GeometricCalculator class:
    - pubic float GetArea(GeometricThing thing);
    - pubic float GetPerimeter(GeometricThing thing);
    - pubic float GetPerimeter(GeometricThing[] thing);
